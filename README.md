# Requirements

* Python (2.6, 2.7)
* Celery (2.2, 3.0, 3.1)
* Django (1.2, 1.3, 1.4, 1.5, 1.6, 1.7)
* django-jsonfield (0.8, 0.9)

django-jsonfield > 0.9.0 does not support django 1.2 and 1.3
