`bpnotify` is notification routing for Django. Application notify function with targets, notify_type, media, extra_data, then bpnotify send notify with backend (example: mail).

Requirements
============

* Python (3.6, 3.9)
* Celery (4.2, 5.1, 5.2)
* Django (2.2, 3.2)
* six
* mock(>=0.7.2)
